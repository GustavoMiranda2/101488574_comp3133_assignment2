## Project Snapshot

This project is an employee management application built for COMP3133 Assignment 2.  
It includes Angular on the frontend, a Node.js GraphQL backend, MongoDB Atlas for persistence, and Docker Compose for local orchestration.

## Stack Used

Frontend: Angular 20, TypeScript, Reactive Forms, Angular Router, Bootstrap  
Backend: Node.js, Express, GraphQL, JWT, Mongoose, Cloudinary  
Database: MongoDB Atlas  
Deployment Platform:  
Local Orchestration: Docker Compose

## Run Locally

```bash
docker compose up --build -d
```

Frontend: http://localhost:4200  
Backend GraphQL: http://localhost:4000/graphql
